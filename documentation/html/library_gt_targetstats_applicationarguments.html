Application Arguments

Library

GenericTarget / Target Stats / Application Arguments

Description

Get all arguments that have been passed to the application. The first argument is the executable name. The maximum number of characters as well as the maximum number of arguments that can be stored in the output are configured via the block mask.

Output Ports

args

N-by-M matrix, where N and M indicate the maximum number of characters per argument and the maximum number of arguments, respectively.
Data Type: uint8

numCharacters

M-dimensional vector, where each element indicates the actual number of characters that represent the argument in the M-th column of the args output matrix.
Data Type: uint32

numArguments

Scalar value indicating the actual number of arguments present in the args output matrix.
Data Type: uint32

Block Parameters

The following table shows all block parameters.
Parameter NameDescription
maxNumCharsPerArgumentThe maximum number of characters per argument.
maxNumArgumentsMaximum number of arguments.
