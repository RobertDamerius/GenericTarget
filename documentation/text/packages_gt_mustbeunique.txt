<h1>GT.mustBeUnique</h1>
Argument validation function to check whether the argument is unique.
If the condition is not met, an assertion fails.

<h2>Syntax</h2>
<pre>GT.mustBeUnique(x)</pre>

<h2>Description</h2>
<code>GT.mustBeUnique(x)</code> checks if the condition is met for the input <code>x</code>.
