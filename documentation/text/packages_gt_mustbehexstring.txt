<h1>GT.mustBeHexString</h1>
Argument validation function to check whether the argument is a hexadecimal string if it's non-empty.
If the condition is not met, an assertion fails.

<h2>Syntax</h2>
<pre>GT.mustBeHexString(x)</pre>

<h2>Description</h2>
<code>GT.mustBeHexString(x)</code> checks if the condition is met for the input <code>x</code>.
