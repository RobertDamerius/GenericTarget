<h1>GT.GetTemplate</h1>
Get a Simulink template model that is pre-configured with all settings for code generation.

<h2>Syntax</h2>
<pre>GT.GetTemplate()</pre>

<h2>Description</h2>
<code>GT.GetTemplate()</code> copies the Simulink template model "GenericTargetTemplate.slx" into the current working directory.
