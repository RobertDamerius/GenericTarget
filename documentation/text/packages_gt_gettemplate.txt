<h1>GT.GetTemplate</h1>
Create a pre-configured template Simulink model and save it to the current working directory.

<h2>Syntax</h2>
<pre>GT.GetTemplate()</pre>

<h2>Description</h2>
<code>GT.GetTemplate()</code> creates or overwrites the Simulink template model "GenericTargetTemplate.slx" in the current working directory.
