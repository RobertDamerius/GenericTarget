<h1>Stop Execution</h1>
<h3>Library</h3>
<span style="font-family: Courier New">GenericTarget / Interface / Stop Execution</span>


<h3>Description</h3>
Stop the execution of the target model. During simulation this block has no effect.


<h2>Input Ports</h2>
<h3>stop</h3>
True if the execution is to be stopped, false otherwise.
<br>
<b>Data Type:</b> <span style="font-family: Courier New">bool</span>
