<h1>Task Overloads</h1>
<h3>Library</h3>
<span style="font-family: Courier New">GenericTarget / Time / Task Overloads</span>


<h3>Description</h3>
Get the current number of <a href="timingscheduling.html#taskoverload">task overloads</a> for the task running at the specified sampletime.
If this task does not exist, zero is returned.
During simulation this block has no effect.


<h2>Output Ports</h2>
<h3>numTaskOverloads</h3>
The number of task overloads that have occurred to this point for the specified task.
<br>
<b>Data Type:</b> <span style="font-family: Courier New">uint64</span>

</table>
