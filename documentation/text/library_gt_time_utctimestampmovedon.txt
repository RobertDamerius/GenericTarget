<h1>UTC Timestamp Moved On</h1>
<h3>Library</h3>
<span style="font-family: Courier New">GenericTarget / Time / UTC Timestamp Moved On</span>


<h3>Description</h3>
If the input UTC timestamp has moved on, then TRUE is returned, otherwise FALSE is returned. The initial condition determines the initial value of the previous input.


<h2>Input Ports</h2>
<h3>timestampUTC</h3>
UTC second of the day in seconds in range [0,86400) (on some systems [0,86401) or [0,86402) to allow leap seconds).
This timestamp can be generated by <a href="library_gt_time_utctimestamp.html">UTC Timestamp</a> block.
<br>
<b>Data Type:</b> <span style="font-family: Courier New">double</span>


<h2>Output Ports</h2>
<h3>movedOn</h3>
TRUE if the input timestamp has moved on, FALSE otherwise.
<br>
<b>Data Type:</b> <span style="font-family: Courier New">bool</span>
