<h1>GT.mustBeCellStr</h1>
Argument validation function to check whether the argument is a cell array of character vectors.
If the condition is not met, an assertion fails.

<h2>Syntax</h2>
<pre>GT.mustBeCellStr(x)</pre>

<h2>Description</h2>
<code>GT.mustBeCellStr(x)</code> checks if the condition is met for the input <code>x</code>.
